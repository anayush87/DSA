# [209. Minimum Size Subarray Sum Solution](https://leetcode.com/submissions/detail/1770036620/)

## Code

```java
class Solution {
    public int minSubArrayLen(int target, int[] nums) {
        int i=0;
        int n=nums.length;
        int sum=0;
        int minlen=Integer.MAX_VALUE;
        for(int j=0;j<n;j++)
        {
            sum+=nums[j];
            while(sum>=target)
            {        
                int len=j-i+1;
                if(len<minlen)
                {
                    minlen=len;
                }
                sum -= nums[i];
                i++;
            }
            

        }
        // if(minlen==Integer.MAX_VALUE)
        // {
        //     return 0;
        // }
        // else
        // {
        //     return minlen;
        // }

        return (minlen==Integer.MAX_VALUE)?0:minlen;
    }
}

# 441. Arranging Coins

**LeetCode Submission:** [Link to Solution](https://leetcode.com/problems/arranging-coins/submissions/1754216673/)

## Code

```java
class Solution {
    public int arrangeCoins(int n) {
       int row = 0;
        int i = 1;
        while(n >= i) {
            n -= i;
            row++;
            i++;
        }
        return row;       
        
    }
}

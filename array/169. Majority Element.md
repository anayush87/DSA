# 169. Majority Element

**LeetCode Submission:** [Link to Solution](https://leetcode.com/problems/majority-element/submissions/1601465626/)

## Code

```java
import java.util.*;
class Solution {
    public int majorityElement(int[] nums) {
        int count = 0;
        int candidate = 0;

        for (int num : nums) {
            if (count == 0) {
                candidate = num;
            }
            count += (num == candidate) ? 1 : -1;
        }

        return candidate;
    }
}

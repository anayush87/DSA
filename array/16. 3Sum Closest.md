# 16. 3Sum Closest

**LeetCode Submission:** [Link to Solution](https://leetcode.com/submissions/detail/1788105513/)

## Code

```java
class Solution {
    public int threeSumClosest(int[] nums, int target) {
        Arrays.sort(nums);
        int n = nums.length;
        int min=nums[0]+nums[1]+nums[2];
        for (int i = 0; i < n - 2;i++ ) 
        {
            int left = i + 1, right = n - 1;

            while (left < right) {
                int sum = nums[i] + nums[left] + nums[right];
                if (Math.abs(sum - target) < Math.abs(min - target)) 
                {
                    min = sum;
                }

                if (sum < target) {
                    left++;
                } else if (sum > target) {
                    right--;
                } else {
                    return target;
                }
            }
        }

        return min;
    }
}
